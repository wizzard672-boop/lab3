# Отчёт по лабораторной работе: Рекурсия и итерация

## 1. Условия задач

### Задача 1. Перестановки длиной k
Функция для вычисления всех перестановок списка длиной `k`.
**Пример:**
k_permutations([1, 2, 3], 2)
[[1, 2], [1, 3], [2, 1], [2, 3], [3, 1], [3, 2]]

Задача 1: Перестановки длиной k
Рекурсивная реализация (k_permutations_recursive):

База: Если k == 0, возвращается пустой список внутри списка [[]] (пустая перестановка). Если k больше длины массива, возвращается пустой список (перестановок нет).

Шаг: Для каждого элемента массива current берётся этот элемент, а из оставшихся элементов (remaining) рекурсивно ищутся перестановки длиной k-1. Затем current добавляется в начало каждой найденной подстановки.

Итеративная реализация (k_permutations_iterative):

Использует встроенную библиотеку itertools, а именно функцию itertools.permutations(arr, k), которая является генератором, написанным на C и работающим по итеративному алгоритму (эквивалентно вложенным циклам).

Результат преобразуется в список списков для соответствия условию.
